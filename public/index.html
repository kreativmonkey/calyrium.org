Install Invoiceninja Archlinux with Nginx

In this tutorial i’ll see how to install Invoice Ninja on a Arch Linux VPS with MariaDB, PHP-FPM and Nginx. Invoice Ninja is a free, open-source solution for invoicing and billing customers and it’s based on Laravel framework. Installing Required Software First you must Login to your System via SSH: ssh user@myVPS Requirements: nginx php-fpm php-mcrypt php-gd php-composer git curl openssl mariadb First update the system: pacman -Syu Setup MariaDB Installing MariaDB (More Information on Archlinux.org ): pacman -S mariadb When the installation is complete, run the following command to secure your installation: mysql_secure_installation Next, we need to create a database for our Invoice Ninja instance.
